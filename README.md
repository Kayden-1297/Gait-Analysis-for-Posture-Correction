Description
This project implements a portable and cost-effective Gait Analysis System using an Arduino Uno microcontroller, HC-SR04 Ultrasonic Sensors, and a buzzer. The system monitors human walking patterns and posture by measuring distance variations during movement. It provides real-time feedback through audio alerts whenever abnormal gait patterns or posture deviations are detected. The project demonstrates embedded systems concepts such as sensor interfacing, real-time data acquisition, signal processing, posture monitoring, and healthcare-oriented assistive technology.

 Working Principle

1. HC-SR04 Ultrasonic Sensors continuously measure distance during walking.
2. Arduino Uno collects and processes sensor readings.
3. Distance values are compared with predefined threshold levels.
4. Any irregular gait movement or posture deviation is detected.
5. The buzzer generates an alert for immediate posture correction.
6. The process repeats continuously for real-time monitoring.

Installation

1. Install Arduino IDE.
2. Connect Arduino Uno to the computer.
3. Connect the HC-SR04 sensor and buzzer according to the circuit diagram.
4. Upload the Arduino source code.
5. Power the system using a 9V battery.
6. Start walking and monitor posture through sensor feedback.
